2.為什麼想成為工程師 ( 請以自我期許帶入：前端、後端、或全端工程師 )
由於自身網站分析師的工作性質，也時常需要碰觸網站的原始碼，例如埋設Google 與 fb追蹤碼，SEO的技術等，雖然平常也會使用GTM來埋設，但偶而客戶會問一些網站的問題，由於自身不是工程師

的網站我都會看一下網頁的原始碼來看看這個網頁的架構是什麼。
